## 一、产品定义

**QuantPrism** 是一个**个人量化投资导航仪**。

> 不是给你看数据的仪表盘。是告诉你下一步该干什么的导航仪。

### 核心用户

- 只有一个用户：你自己
- 有投资经验，量化新手
- 不写代码，AI 代劳
- 需要透明度：看清每个决策背后的数据和逻辑

### 设计原则

1. **行动导向** — 每个页面回答一个问题："我现在该做什么？"
2. **过程透明** — 不只给结论，展示得出结论的每一步数据
3. **指标翻译** — 每个专业术语旁边有通俗解释
4. **AI 驱动** — 搜索策略、生成策略、扫描标的、给建议，都靠 AI

### 功能 1：目标设定

**用户问题：** "我想要什么？"

**交互方式：** 单页表单。4 个目标维度全部在一个页面上，可以只填 1 个也可以全填。每个字段旁有 (i) 图标，hover 显示通俗解释。

```
┌────────────────────────────────────────────────────────────┐
│  设定投资目标                                    [保存目标]  │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  以下目标至少填 1 个，也可以全部填写同时生效。               │
│                                                            │
│  ┌─ 收益目标 (i) ──────────────────────────────────────┐  │
│  │  年化收益率:  ●───────────────○  15%                 │  │
│  │  提示: 10万投入，年底变 11.5 万。标普长期约 10%。     │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                            │
│  ┌─ 回撤目标 (i) ──────────────────────────────────────┐  │
│  │  最大可接受回撤:  ●──────○  10%                      │  │
│  │  提示: 10 万的账户最多跌到 9 万。                     │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                            │
│  ┌─ 交易标的 (i) ──────────────────────────────────────┐  │
│  │  [✅] 美股股票    [✅] 美股 ETF                       │  │
│  │  [✅] 股票期权    [ ] 加密货币                        │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                            │
│  ┌─ 持仓周期 (i) ──────────────────────────────────────┐  │
│  │  [日内]  [几天~几周]  [几周~几月]  [几月~一年]        │  │
│  └─────────────────────────────────────────────────────┘  │
│                                                            │
│  ─── 系统解读（填完自动生成）──────────────────────────── │
│  基于你的目标，系统推导出:                                  │
│  • 需要 Sharpe > 1.0 的策略 (i)                            │
│  • 最大同时持仓建议 3-5 个                                  │
│  • 单笔最大亏损建议控制在 2% 以内                           │
│                                                            │
│                                            [保存并找策略 →] │
└────────────────────────────────────────────────────────────┘
```

**规则：** 至少填 1 个目标才能保存。可以只设收益目标不设回撤，也可以只设回撤不设收益。系统解读根据已填目标自动推导。

### 功能 2：策略猎手（Strategy Hunter）

**用户问题：** "有什么策略能帮我实现目标？"

**两个来源：**
- **搜索现有策略**：GitHub、量化论坛、学术论文、已知策略库

### 功能 6：风控护盾 + 全局配置

**用户问题：** "万一亏了怎么办？怎么防守？"

**风控策略 vs 交易策略的关系：**

```
交易策略 = 告诉你「什么时候买卖」（进攻）
  例: 双均线交叉买入、RSI 超卖抄底

风控策略 = 告诉你「什么时候必须停手」（防守）
  例: 单笔亏 2% 止损、总回撤 >10% 全部清仓、VIX>30 暂停

关系: 风控策略是交易策略的"保险丝"，永远优先级更高。
      交易策略说"买"，但风控策略说"不行，风险超标" → 不买。
      回测时风控策略也参与计算（加止损 vs 不加的区别）。
```

## 四、指标翻译词典

**展示方式：** 指标名称后显示 (i) 图标，鼠标 hover 弹出通俗解释 tooltip，移开消失。不额外占用表格空间。

所有页面中出现的专业指标都附带翻译：

| 指标 | 通俗翻译 |
|------|---------|
| 年化收益 X% | 平均每年赚 X%，10 万变 10*(1+X%) 万 |
| 最大回撤 -X% | 最惨时从最高点跌了 X%，10 万变 10*(1-X%) 万 |
| Sharpe Ratio X | 每承受 1 块钱风险赚 X 块。>1.0 好，>1.5 很好，>2.0 优秀 |
| 胜率 X% | 100 次交易有 X 次赚钱 |
| 盈亏比 X | 赚的时候平均赚的是亏的 X 倍 |
| VIX | 恐慌指数。<15 很平静，15-20 正常，20-30 紧张，>30 恐慌 |
| Delta X | 股票涨 $1 期权涨 $X。0.5 = 跟涨一半 |
| IV X% | 隐含波动率。市场预期这个股票未来 X% 范围内波动 |
| RSI | 超买超卖指标。>70 太热了可能要跌，<30 太冷了可能要涨 |
| SMA/EMA | 均线。过去 N 天的平均价格。价格在均线上方 = 趋势向上 |
| Calmar Ratio | 年化收益 / 最大回撤。越大越好。>0.5 及格，>1.0 优秀 |
| 连败次数 | 连续亏损的次数。越多说明策略越不稳定 |
| SQN 评分 | 系统质量数。<1.6 很差，1.6-2.0 一般，2.0-2.5 良好，2.5-3.0 优秀，>3.0 卓越 |
| Sortino Ratio | 类似 Sharpe，但只惩罚向下波动。Sortino > Sharpe = 策略的波动主要是向上的 |
| Omega Ratio | 赚的概率加权 / 亏的概率加权。>1 = 正期望，越大越好 |
| VaR 95% | 正常情况下，100天中最坏5天的单日亏损下限 |
| CVaR 95% | 那5%最坏情况的平均亏损，比 VaR 更保守的风险指标 |
| PSR | 概率夏普率。告诉你当前 Sharpe 是否只是运气。>95% 才算统计显著 |
| MAE | 最大逆向回撤。这笔交易持仓期间最惨时亏了多少（入场后的最大浮亏） |
| MFE | 最大顺向收益。这笔交易持仓期间最赚时赚了多少（入场后的最大浮盈） |
| Kelly比例 | 数学最优单笔仓位。Full Kelly 波动大，实际用 Half Kelly（除以2）|
| 偏度 | 收益分布的对称性。>0 = 右偏（有大赚机会）<0 = 左偏（有大亏风险）|
| 峰度 | 收益分布的"肥尾"程度。>3 = 极端事件比正态分布更频繁 |
| 平行时空胜率 | 在过去 N 年中，随机入场持仓 X 天，最终盈利的概率 |
